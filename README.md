# AI-Assisted Ransomware Detection Framework (BTRDF)

> AI-assisted ransomware detection and recovery framework using ANN models with Microsoft Defender and Sentinel integration

## Overview
This project presents a Blue Team Ransomware Defense Framework (BTRDF) that integrates Artificial Neural Network (ANN)–based anomaly detection with SIEM tools to improve ransomware detection and response.

## Key Features
- ANN-based anomaly detection
- SIEM integration (Microsoft Defender & Sentinel)
- Simulated ransomware attack environment
- Automated response actions
- Risk scoring and threat classification
